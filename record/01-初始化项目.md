## 安装
使用create-react-app安装，考虑到全面学习，所以使用了typescript，在使用create-react-app安装后，我将项目删除，再次安装使用了typescript模板，详情 `02使用typescript.md`
```
npx create-react-app netease_cloud_music
cd netease_cloud_music
npm start
```
